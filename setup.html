How to Use Website Blocker

1
Add websites to your block list

Enter domain names of websites you want to block in the options page.

2
Set blocking duration

Choose how long you want the websites to be blocked (in minutes).

3
Activate blocking

Click the "Activate Blocking" button to start blocking the websites for your specified duration.

4
Stay focused

If you try to visit a blocked website, you'll see a blocking image and the tab will close automatically.

You can access the options page anytime by clicking on the extension icon and selecting "Options".
